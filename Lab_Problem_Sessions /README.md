# CMPE 140 - Laboratory Problem Sessions (PS)

This repository contains the core programming exercises and laboratory tasks developed for the **CMPE 140 - Computer Programming 1** course at **Kadir Has University**. The codes demonstrate fundamental and intermediate C++ programming concepts applied to engineering logic problems.

## 📂 Contents & Technical Overview

### 1. Quantity Discount Calculator (`LAB-PS-02.cpp`)
Calculates dynamic pricing and discount percentages based on user input quantities.
* **Key Concepts:** Standard I/O operations, robust input validation (negative value filtering), and sequential `if-else if` control structures for numerical range evaluation.

### 2. University Admission Evaluator (`LAB-PS-03.cpp`)
Evaluates academic and disciplinary metrics against specific university admission criteria.
* **Key Concepts:** Complex Boolean logic matrices, logical operators (`AND`, `OR`), state management using `bool` data types, and exhaustive conditional output generation.

### 3. Robot Battery Management System (`LAB-PS-07.cpp`)
Manages and analyzes the battery statuses of a fleet of robots using 1D arrays.
* **Key Concepts:** Modular architecture via custom functions (`printBattery`, `countAbove`, `chargeAll`), array manipulation through iterative loops, and data integrity enforcement using `const` parameters to prevent unintended array mutation.

---

*Compiled and tested using standard C++ compilers (GCC/MSVC).*

# CMPE 140 - Laboratuvar Uygulama Çalışmaları (PS)

Bu dizin, **Kadir Has Üniversitesi CMPE 140 - Bilgisayar Programlama 1** dersi süresince haftalık laboratuvar oturumlarında (Problem Sessions) geliştirilen temel programlama egzersizlerini içermektedir. Kodlar, mühendislik mantığı problemlerine uygulanan temel ve orta düzey C++ programlama kavramlarını göstermektedir.

## 📂 İçerik ve Teknik Özet

### 1. Miktar İndirimi Hesaplayıcı (`LAB-PS-02.cpp`)
Kullanıcının girdiği miktara göre dinamik fiyatlandırma ve indirim oranları hesaplar.
* **Temel Kavramlar:** Temel girdi/çıktı işlemleri, negatif değerleri engelleyen girdi kontrolü ve sayısal aralıkları süzmek için sıralı `if-else if` kontrol yapıları.

### 2. Üniversite Kabul Değerlendirici (`LAB-PS-03.cpp`)
Akademik ve disiplin kriterlerini belirli üniversite kabul şartlarına göre değerlendirir.
* **Temel Kavramlar:** Karmaşık Boolean mantık matrisleri, mantıksal operatörler (`&&`, `||`), `bool` veri tipiyle durum yönetimi ve tüm kabul/ret kombinasyonlarının çıktı üretimi.

### 3. Robot Batarya Yönetim Sistemi (`LAB-PS-07.cpp`)
Bir robot filosunun batarya durumlarını tek boyutlu diziler (array) kullanarak yönetir ve analiz eder.
* **Temel Kavramlar:** Özel fonksiyonlar (`printBattery`, `countAbove`, `chargeAll`) ile modüler mimari, döngülerle dizi manipülasyonu ve verilerin korunması için `const` parametre kullanımı.

---

*Standart C++ derleyicileri (GCC/MSVC) kullanılarak derlenmiş ve test edilmiştir.*
