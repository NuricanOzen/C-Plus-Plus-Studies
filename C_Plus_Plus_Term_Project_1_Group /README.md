# 🇬🇧 1913 US Income Tax Calculator

Welcome! 👋

This project is an automated tax calculator developed for the **CMPE 140 - Computer Programming 1** course at **Kadir Has University**. It calculates the exact income tax based on the original U.S. income tax schedule of 1913.

**Project Features & Logic:**
* **Progressive Calculation:** Accurately applies 6 different tax rates (from 1% to 6%) to their respective income segments using hardcoded constant parameters.
* **Continuous Testing:** Utilizes a `do-while` loop, allowing the user to test multiple income intervals continuously without restarting the executable.
* **Input Validation:** Prevents logical execution errors by automatically rejecting negative integer inputs.

**Team Members:** Nurican Özen, Fatih Kemal Turhan, Akın Demirbaş, Ertuğrul Emre Köksal

# 🇹🇷 1913 ABD Gelir Vergisi Hesaplayıcı

Hoş geldiniz! 👋

Bu proje, **Kadir Has Üniversitesi CMPE 140 - Bilgisayar Programlama 1** dersi kapsamında geliştirilmiş otomatik bir vergi hesaplayıcıdır. 1913 yılı orijinal ABD gelir vergisi tarifesine göre kademeli vergi hesaplaması yapmaktadır.

**Proje Özellikleri ve Algoritma Mantığı:**
* **Kademeli Hesaplama:** Sabit (const) parametreler kullanarak, geliri 6 farklı dilime (%1'den %6'ya kadar) ayırır ve ilgili dilimlere doğru vergi oranlarını uygular.
* **Sürekli Test İmkanı:** `do-while` döngüsü sayesinde, programı yeniden başlatmaya gerek kalmadan arka arkaya farklı gelir değerleri için test yapma olanağı sunar.
* **Girdi Kontrolü (Input Validation):** Kullanıcının negatif değer girmesini engelleyerek algoritmanın çökmesini veya mantıksal hata vermesini engeller.

**Takım Üyeleri:** Nurican Özen, Fatih Kemal Turhan, Akın Demirbaş, Ertuğrul Emre Köksal
